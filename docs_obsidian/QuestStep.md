# QuestStep (этап квеста)

Таблица: `quest_steps` · Модель: `backend/app/models/quest_step.py`

Подзадача внутри [[Quest|квеста]]. Этапы упорядочены по `sort_order` и могут давать частичный XP.

## Связи

- **Родитель:** [[Quest]] (`quest_id`, обязательно)
- **Награды:** частичный XP суммируется при завершении квеста — см. [[Economy]]

## Поля

| Поле | Тип | Назначение |
|------|-----|------------|
| `title` | string(200) | Название этапа |
| `description` | text | Описание шага |
| `sort_order` | int | Порядок в списке этапов |
| `status` | enum | `pending` · `completed` · `skipped` |
| `is_optional` | bool | Не блокирует завершение [[Quest\|квеста]] |
| `xp_reward` | int | Частичный опыт за этап |
| `completed_at` | datetime | Когда этап выполнен |
| `created_at` | datetime | Когда этап создан |

## Статусы (`status`)

| Значение | Описание |
|----------|----------|
| `pending` | Ещё не выполнен |
| `completed` | Выполнен |
| `skipped` | Пропущен (для опциональных этапов) |

## Логика завершения квеста

- Все **обязательные** этапы (`is_optional = false`) должны быть `completed` или `skipped`
- Опциональные этапы не блокируют переход [[Quest]] в `completed`
- При провале [[Quest]] (`failed`) XP за выполненные этапы может быть начислен частично — см. [[Economy]]
