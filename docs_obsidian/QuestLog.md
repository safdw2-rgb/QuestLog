# QuestLog — справочник схемы данных

RPG-задачник: квесты, этапы, карта, дневник и достижения.

> Исходный документ: `docs/SCHEMA.md`  
> Код моделей: `backend/app/models/`

## Сущности

| Сущность | Описание |
|----------|----------|
| [[Adventurer]] | Профиль игрока: XP, золото, уровень |
| [[Quest]] | Центральная сущность — квестовое задание |
| [[QuestStep]] | Этапы / подзадачи внутри [[Quest\|квеста]] |
| [[Faction]] | Фракция / категория квестов |
| [[Location]] | Точка на карте |
| [[Achievement]] | Шаблон достижения |
| [[JournalEntry]] | Запись в квестовом дневнике |

## Сквозные темы

- [[Economy]] — награды, XP, золото, level-up
- [[ER-диаграмма]] — связи между таблицами

## Быстрая навигация по связям

```
[[Adventurer]] ──владеет──► [[Quest]] ──содержит──► [[QuestStep]]
      │                         │
      │                    привязан к
      │                         ├── [[Faction]]
      │                         ├── [[Location]]
      │                         └── [[JournalEntry]]
      │
      ├── разблокирует ──► [[Achievement]]
      └── пишет ──► [[JournalEntry]]
```
