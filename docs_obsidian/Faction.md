# Faction (фракция)

Таблица: `factions` · Модель: `backend/app/models/faction.py`

Категория / фракция квестов: «Работа», «Здоровье», «Дом» и т.д. Используется для фильтров в дневнике и в условиях [[Achievement|достижений]].

## Связи

- **Квесты фракции:** [[Quest]] (`faction_id`, опционально)
- **Достижения:** условие `faction_quests` ссылается на `slug` фракции

## Поля

| Поле | Тип | Назначение |
|------|-----|------------|
| `name` | string(64) | Отображаемое имя («Здоровье») |
| `slug` | string(64) | Уникальный идентификатор (`health`, `work`, `home`) |
| `icon` | string(32) | Иконка в UI |
| `color` | string(7) | Цвет в формате `#RRGGBB` |
| `description` | text | Описание фракции |

## Примеры фракций

| name | slug | Смысл |
|------|------|-------|
| Работа | `work` | Профессиональные [[Quest\|квесты]] |
| Здоровье | `health` | Спорт, врачи, сон |
| Дом | `home` | Быт, ремонт, уют |

## Достижения по фракциям

Пример условия в [[Achievement]]:

```json
{"faction_slug": "health", "count": 5}
```

Означает: завершить 5 квестов фракции «Здоровье».
